# YelpCamp

YelpCamp is a Full Stack NodeJs Web App Project that was completed as a part of [Colt Steele's bootcamp](https://www.udemy.com/the-web-developer-bootcamp/).

Built with: NodeJS, ExpressJS, Heroku, MongoDB at mLab
